Robot Framework Commands Test

Test Instructions

This page tests if Robot Framework commands are working in the Testoriumz Recorder.

  1. Open the Testoriumz Recorder extension
  2. Start recording on this page
  3. Try to use Robot Framework commands like "Go To", "Click Element", "Input Text"
  4. Check if the commands appear in the command dropdown
  5. Try to execute the commands

Test Elements

Use these elements to test Robot Framework commands:

Expected Robot Framework Commands

These commands should now be available in the recorder:

Navigation Commands:

  • Go To - Navigate to URL
  • Go Back - Go back in browser history
  • Reload Page - Refresh the current page

Element Interaction Commands:

  • Click Element - Click on any element
  • Click Link - Click on a link by text
  • Input Text - Type text into input field
  • Input Password - Type password into password field
  • Select From List By Label - Select from dropdown
  • Submit Form - Submit a form

Wait Commands:

  • Wait Until Element Is Visible - Wait for element
  • Wait Until Page Contains - Wait for text
  • Sleep - Wait for specified time

Assertion Commands:

  • Page Should Contain - Assert page contains text
  • Element Should Be Visible - Assert element visible
  • Element Should Be Enabled - Assert element enabled

Test Steps

Try recording these steps:

  1. Use Go To to navigate to this page
  2. Use Input Text to type in the test input field
  3. Use Click Element to click the test button
  4. Use Select From List By Label to select an option
  5. Use Page Should Contain to verify text is present

Debug Information

Check the browser console for debug information about Robot Framework commands.

Look for messages like:

  • "Robot Framework commands loaded: [list of commands]"
  • "Total commands available: [number]"
